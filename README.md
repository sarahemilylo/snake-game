# 🐍 Snake Game (Python)

![Python](https://img.shields.io/badge/Python-3.x-blue.svg)
![Library](https://img.shields.io/badge/Library-Turtle-green.svg)
![Status](https://img.shields.io/badge/Status-Completed-brightgreen.svg)

A simple implementation of the classic Snake game using Python’s built-in `turtle` graphics library. This project demonstrates core programming concepts such as loops, event handling, and collision detection in a visual and interactive way.

## 🚀 Features

- Smooth real-time snake movement  
- Dual controls (WASD + Arrow keys)  
- Random food spawning  
- Score and high score tracking  
- Increasing difficulty over time  
- Collision detection:
  - Wall collision resets game  
  - Self collision resets game  

---

## 🎯 Controls

| Action | Key |
|------|-----|
| Move Up | W / ↑ |
| Move Down | S / ↓ |
| Move Left | A / ← |
| Move Right | D / → |
